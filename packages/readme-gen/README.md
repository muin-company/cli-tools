# @muin/readme-gen

Generate professional, comprehensive README files from your project with an interactive CLI

![npm version](https://img.shields.io/npm/v/@muin/readme-gen.svg) ![npm downloads](https://img.shields.io/npm/dm/@muin/readme-gen.svg) ![license](https://img.shields.io/badge/license-MIT-blue.svg) ![node version](https://img.shields.io/badge/node-%3E%3D16.0.0-green.svg)


## Installation

### As a CLI tool

```bash
npm install -g @muin/readme-gen
```

Or use with npx:

```bash
npx @muin/readme-gen
```

### As a library

```bash
npm install @muin/readme-gen
```
