# Changelog

All notable changes to this project will be documented in this file.

## [0.1.2] - 2026-03-27

### Fixed
- Package metadata improvements (keywords, repository links)

## [0.1.1] - 2026-03-26

### Fixed
- Minor bug fixes and stability improvements

## [0.1.0] - 2026-03-26

### Added
- Multi-file support — analyze multiple files in one command
- Function-level analysis with `--function` flag
- JSON output mode for scripting and CI integration
- Verbose mode with full commit details
- Line range support (`file:start-end`)

### Improved
- Better AI prompts for more accurate explanations
- Cleaner output formatting with Chalk

## [0.0.1] - 2026-03-25

### Added
- Initial release
- AI-powered git history explanation using Claude
- Git blame + git log analysis pipeline
- Single file and line-level targeting
- Colorized terminal output
- `--help` and `--version` flags
